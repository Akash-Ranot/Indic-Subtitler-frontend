# Indic-Subtitler Frontend

## Overview
This repository contains the frontend for the Indic-Subtitler, an open-source subtitling platform designed to transcribe videos/audios in Indic languages and translate subtitles using ML models.

## Objective
The frontend is designed to provide a user-friendly interface for uploading videos/audios, displaying transcriptions, and downloading subtitles in various Indic languages.

## Technology Stack
The frontend is built using Next.js, a React framework that offers features such as file-based routing and API routes. For styling, we use either Tailwind CSS or styled-components.

## Setup and Installation
Install node modules

use npm run dev for starting the project
